## 语法
1.标题以#开头

    # 一级标题

2.字体
 
 斜体用*(星号)包起来 *这是斜体*

 粗体用**包起来 **这是粗体**

 斜体加粗***包起来 ***粗斜体***

 删除线~~包起来 ~~删除字~~

3.引用

以>开始的，可以多个嵌套
>这是引用的内容
>>这是引用的内容
>>>>>>>>>>这是引用的内容

4.分割线

三个或者三个以上的 - 或者 * 都可以。

9.代码

单行代码：代码之间分别用一个反引号包起来

`var a = 1`

代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行

```
var a = 3;
var b = 3;
console.log(a,b)
```
